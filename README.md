## ✨ Мүмкіндіктер

- 🗺️ **Интерактивті карта** — Leaflet арқылы аудандарды визуализациялау
- 📊 **KPI Dashboard** — жылу жүктемесі, қысым, тозу деңгейі
- 🤖 **AI Чат-бот** — Gemini AI арқылы қазақша/орысша жауап
- 📈 **Chart.js графиктер** — нақты уақытта жаңарып тұратын деректер
- 🏘️ **5 аудан** — Абай, Әл-Фараби, Еңбекші, Қаратау, Тұран
- 📨 **Азаматтар арызы** — шұғыл хабарламалар жүйесі
- 🌐 **Three.js 3D** — желі визуализациясы
